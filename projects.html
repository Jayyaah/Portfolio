🎨 Passe P'Art Où !

Description :
Passe P'Art Où ! est un prototype d’application iOS axé sur la découverte artistique, inspiré du concept de Pokémon Go.

L’application permet aux utilisateurs de :
🗺️ Explorer une carte interactive avec différents lieux d’art (théâtres, musées, galeries, etc.)
🎯 Réaliser des missions liées à ces lieux pour débloquer des récompenses
🎁 Échanger ces récompenses dans une boutique virtuelle contre divers items
🎨 Personnaliser leur interface utilisateur avec les items gagnés

Fonctionnalités principales :
🗺️ Carte interactive : visualisation des lieux d’art autour de soi
🎯 Missions : défis à accomplir dans différents endroits culturels
🎁 Récompenses : système de gains et échanges dans la boutique intégrée
✨ Personnalisation : customisation de l’interface grâce aux items obtenus

🛠️ Technologies utilisées :
Swift / SwiftUI
MapKit pour la carte interactive
Architecture MVVM (modèle-vue-vue modèle)
Gestion des données locales
